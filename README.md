## 🧠 Overview

VisuAlgorithm transforms the way you learn Data Structures and Algorithms. Instead of staring at static pseudocode, watch your algorithms think in real time — step by step, node by node, frame by frame.

Built as a full-featured academic project using solid Object-Oriented Programming (OOP) principles, VisuAlgorithm lets you construct data structures interactively, manipulate them on a live canvas, and watch algorithms execute with smooth, adjustable animations.

## 🚀 Quick Start (No Code Required)

If you just want to use the application to learn algorithms, you don't need to download the source code or install an IDE. Download the pre-built application from our [Releases Page](https://github.com/SayemTanvir/Algorithm_Visualizer/releases).

### 📦 Portable Installation (.zip)

The easiest way to run the app without a formal installation process.

1. Download the **VisuAlgorithm-Portable.zip** file from the Releases page
2. Extract the .zip file to your preferred location (e.g., your Desktop or C:\Programs\VisuAlgorithm)
3. Open the extracted folder
4. **To Run:** Double-click the **VisuAlgorithm.exe** (Windows) or the provided run script
5. **Create a Desktop Shortcut:** Right-click the .exe file inside the extracted folder, select Show more options → Send to → Desktop (create shortcut)

### 🪟 Windows Installation (.exe or .msi)

1. Go to the [Releases Page](https://github.com/SayemTanvir/Algorithm_Visualizer/releases) and download **VisuAlgorithm-Setup.exe**
2. Double-click the downloaded file to run the installer
3. Follow the installation wizard
4. Check the box to "Create a Desktop Shortcut" during installation
5. Once installed, simply double-click the VisuAlgorithm icon on your desktop to launch the app!

**Note:** If Windows SmartScreen blocks the app, click "More Info" → "Run Anyway"

### 🍏 macOS Installation (.app or .dmg)

1. Go to the [Releases Page](https://github.com/SayemTanvir/Algorithm_Visualizer/releases) and download **VisuAlgorithm.dmg**
2. Double-click the .dmg file to mount it
3. Drag and drop the **VisuAlgorithm.app** icon into your **Applications** folder
4. **Creating a Desktop Shortcut (Alias):** Right-click the app in your Applications folder and select "Make Alias", then drag the alias to your desktop
5. **Running for the first time:** macOS Gatekeeper may block the app since it's from an "unidentified developer"
   - Right-click (or Control-click) the app and select **Open**
   - Click **Open** again in the warning dialog box to whitelist the application

## 👨‍💻 Developer Guide (Source Code)

Want to contribute, view the code, or build it yourself? Follow these steps to set up your development environment.

### ✅ Prerequisites

- **JDK 21 or higher** (Ensure JAVA_HOME is set)
- **JavaFX SDK 21+** (Matches your JDK version)
- **Git** installed on your system (optional if using ZIP download)

### 📥 1. Get the Source Code

Choose one of the following methods to get the project files:

**Option A (Git):** Clone the repository via terminal:
```bash
git clone https://github.com/SayemTanvir/Algorithm_Visualizer.git
cd Algorithm_Visualizer
```

**Option B (ZIP):** Click the green "Code" button at the top of the GitHub page, select "Download ZIP", and extract the folder to your computer.

### 🧠 2. IntelliJ IDEA (Recommended)

1. **Open the Project:** File > Open > Select the extracted/cloned Algorithm_Visualizer folder
2. **Set the JDK:** Go to File > Project Structure > Project and ensure SDK is set to Java 21
3. **Add JavaFX Library:**
   - Go to File > Project Structure > Libraries
   - Click + > Java and locate your downloaded JavaFX SDK lib folder
4. **Configure Run/Debug:**
   - Click Run > Edit Configurations
   - Add a new Application configuration
   - Set Main Class to Launcher (or HelloApplication)
5. **Crucial Step:** In the VM Options (Modify options > Add VM options), paste the following, replacing the path with your actual JavaFX lib path:
```
--module-path "C:\path\to\javafx-sdk-21\lib" --add-modules javafx.controls,javafx.fxml
```
6. **Click Run** (Shift + F10)

### 💻 3. VS Code Setup

1. **Install Extensions:** Install the Extension Pack for Java by Microsoft
2. **Open the Folder:** File > Open Folder > Select Algorithm_Visualizer
3. **Configure launch.json:**
   - Go to the Run and Debug tab (Ctrl+Shift+D) and click "create a launch.json file"
   - Add the vmArgs to point to your JavaFX library:

```json
{
    "type": "java",
    "name": "Run VisuAlgorithm",
    "request": "launch",
    "mainClass": "com.yourpackage.Launcher",
    "projectName": "Algorithm_Visualizer",
    "vmArgs": "--module-path \"/path/to/javafx-sdk-21/lib\" --add-modules javafx.controls,javafx.fxml"
}
```

4. **Press F5** to launch the application
