## рҹ§  Overview

**VisuAlgorithm** transforms the way you learn Data Structures and Algorithms. Instead of staring at pseudocode, watch your algorithms **think** in real time вҖ” step by step, node by node, frame by frame.

Built as a full-featured academic project using solid **Object-Oriented Programming** principles, VisuAlgorithm lets you:

- рҹҸ—пёҸ &nbsp; **Construct** data structures from scratch interactively
- рҹҺ® &nbsp; **Interact** with them dynamically through a live canvas
- рҹҺ¬ &nbsp; **Watch** algorithms execute step-by-step with smooth animations

## рҹ“Ұ Getting Started

### вң… Prerequisites

Make sure you have the following installed:

```bash
вң” Java Development Kit (JDK 17+)
вң” JavaFX SDK
вң” IntelliJ IDEA / Eclipse / VS Code
```

### вҡҷпёҸ Installation

**1. Clone the repository**
```bash
git clone https://github.com/your-username/VisuAlgorithm.git
cd VisuAlgorithm
```

**2. Open in your IDE**
```
IntelliJ IDEA  вҶ’  File > Open > Select project folder
Eclipse        вҶ’  File > Import > Existing Project
VS Code        вҶ’  Open folder, install Java Extension Pack
```

**3. Configure JavaFX SDK**
```
Add JavaFX libraries to your module path / classpath
Set VM options:  --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml
```

**4. Run the application**
```bash
# Entry points:
src/main/java/Launcher.java
# or
src/main/java/HelloApplication.java
```
